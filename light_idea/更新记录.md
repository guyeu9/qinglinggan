## 2026-03-12 - 路由导航系统

### 新增功能
- 集成 `go_router` 包构建声明式路由系统
- 创建路由名称常量管理，便于 IDE 自动补全
- 添加路由导航扩展方法，简化页面跳转代码

### 新建文件

#### 1. `lib/core/router/route_names.dart`
- 定义 `RouteNames` 类：包含所有路由名称常量
  - `home` - 首页
  - `ideaDetail` - 灵感详情页
  - `recycleBin` - 回收站
  - `dataManagement` - 数据管理
  - `settings` - 设置
  - `aiHub` - AI灵感中心
  - `aiSettings` - AI设置
  - `help` - 帮助
- 定义 `RoutePaths` 类：包含所有路由路径常量
  - 提供 `ideaDetailPath(String id)` 辅助方法构建详情页路径

#### 2. `lib/core/router/app_router.dart`
- 配置 `GoRouter` 实例 `appRouter`
- 实现8个路由配置：
  | 路径 | 名称 | 说明 |
  |------|------|------|
  | `/` | home | 首页 |
  | `/idea/:id` | ideaDetail | 灵感详情页（带路径参数） |
  | `/recycle-bin` | recycleBin | 回收站 |
  | `/data-management` | dataManagement | 数据管理 |
  | `/settings` | settings | 设置 |
  | `/ai-hub` | aiHub | AI灵感中心 |
  | `/ai-settings` | aiSettings | AI设置 |
  | `/help` | help | 帮助 |
- 创建 `PlaceholderPage` 占位符页面，包含：
  - 页面标题和图标展示
  - 导航测试按钮（便于开发调试）
  - 返回按钮支持
- 添加 `GoRouterExtension` 扩展方法：
  - `goToHome()` - 跳转到首页
  - `goToIdeaDetail(String id)` - 跳转到灵感详情
  - `goToRecycleBin()` - 跳转到回收站
  - `goToDataManagement()` - 跳转到数据管理
  - `goToSettings()` - 跳转到设置
  - `goToAiHub()` - 跳转到AI灵感中心
  - `goToAiSettings()` - 跳转到AI设置
  - `goToHelp()` - 跳转到帮助

### 修改文件

#### `lib/main.dart`
- 导入 `core/router/app_router.dart`
- 替换 `MaterialApp` 为 `MaterialApp.router`
- 配置 `routerConfig: appRouter`
- 移除原有的 `HomePage` 类定义
- 添加 `debugShowCheckedModeBanner: false`

### 使用示例

```dart
// 使用路径导航
context.go('/idea/123');

// 使用命名路由导航
context.goNamed(RouteNames.ideaDetail, pathParameters: {'id': '123'});

// 使用扩展方法导航
context.goToIdeaDetail('123');
context.goToSettings();

// 返回上一页
context.pop();
```

## 2026-03-12 - 数据管理页面

### 新增功能
- 根据原型图 5数据管理 实现数据管理页面
- 支持暗黑模式，使用 AppColors 颜色系统
- 数据导出功能（Excel、数据库格式）
- 数据导入功能（文件上传区域，支持点击和拖拽交互）
- 数据安全提示栏

### 新建文件

#### 1. `lib/presentation/pages/data_management/data_management_page.dart`
- 实现 `DataManagementPage` 有状态组件
- 页面结构：
  - **顶部导航栏**：返回按钮、标题"数据管理"
  - **数据导出区**：
    - 导出至Excel卡片（.xlsx格式）- 绿色成功色图标
    - 导出至数据库卡片（.db格式）- 蓝色信息色图标
    - 点击显示确认对话框
  - **数据导入区**：
    - 文件上传区域（支持点击和鼠标悬停效果）
    - 支持 .xlsx, .db 备份文件格式提示
    - 导入说明：合并数据，不会覆盖现有内容
  - **数据安全提示栏**：
    - 警告色边框和背景
    - 安全提示图标和文字说明
- 交互功能：
  - 导出确认对话框
  - 导出成功提示（SnackBar）
  - 文件上传区域悬停效果

### 修改文件

#### `lib/core/router/app_router.dart`
- 导入 `DataManagementPage`
- 将数据管理路由从 `PlaceholderPage` 替换为 `DataManagementPage`

### 页面特性

| 功能区域 | 说明 |
|---------|------|
| 数据导出 | 支持 Excel (.xlsx) 和数据库 (.db) 两种格式 |
| 数据导入 | 点击或拖拽上传，支持 .xlsx 和 .db 文件 |
| 安全提示 | 提醒用户妥善保管导出文件，定期备份 |
| 暗黑模式 | 完整支持浅色/深色主题切换 |
