**1. 项目结构创建**
- 创建 Flutter 项目 `light_idea`（组织标识：com.lightidea）
- 建立四层架构目录结构：
  - `core/` - 核心工具层
  - `domain/` - 领域层（业务实体和仓库接口）
  - `data/` - 数据层（Isar 模型和仓库实现）
  - `application/` - 应用层（业务用例）

**2. Core 层实现**
- `app_constants.dart` - 应用常量定义（内容限制、分页配置、AI 配置等）
- `result.dart` - 结果封装类（Success/Error 密封类）
- `app_logger.dart` - 日志工具（基于 logger 包）
- `date_formatter.dart` - 日期格式化工具（支持相对时间显示）
- `text_helper.dart` - 文本工具类（截断、验证、中文字符统计）

**3. Domain 层实现**
- `idea.dart` - 灵感实体（IdeaEntity）+ AIStatus 枚举
- `category.dart` - 分类实体（CategoryEntity）
- `tag.dart` - 标签实体（TagEntity）
- `idea_repository.dart` - 灵感仓库接口（完整 CRUD + 软删除）
- `category_repository.dart` - 分类仓库接口
- `tag_repository.dart` - 标签仓库接口

**4. Data 层实现**
- `idea_model.dart` - 灵感 Isar 数据模型（含 toEntity/fromEntity）
- `category_model.dart` - 分类 Isar 数据模型
- `tag_model.dart` - 标签 Isar 数据模型
- `isar_database.dart` - Isar 数据库初始化类
- `seed_data.dart` - 种子数据（三大分类）
- `idea_repository_impl.dart` - 灵感仓库实现
- `category_repository_impl.dart` - 分类仓库实现
- `tag_repository_impl.dart` - 标签仓库实现

**5. Application 层实现**
- `create_idea.dart` - 创建灵感用例（含数据验证）
- `update_idea.dart` - 更新灵感用例
