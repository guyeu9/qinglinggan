## \[v0.5.3] - 2026-03-13

### 📥 数据导入服务

#### 新增功能

**ImportService 导入服务** ✅

| 文件 | 变更类型 | 说明 |
| --- | --- | --- |
| `lib/application/services/import_service.dart` | 新建 | 数据导入服务 |

**核心功能**:

1. **ConflictStrategy 枚举** - 冲突处理策略
   - `overwrite` - 覆盖已存在的记录
   - `skip` - 跳过已存在的记录
   - `merge` - 合并记录（保留原有数据，补充新数据）

2. **ImportResult 类** - 导入结果
   - `successCount` - 成功导入数量
   - `skipCount` - 跳过数量
   - `errorCount` - 错误数量
   - `errors` - 错误信息列表

3. **importFromExcel 方法** - Excel 导入
   - 支持自定义列名映射（id, content, category, tags, createdAt）
   - 自动解析表头
   - 支持日期类型和字符串日期解析
   - 自动创建分类和标签

4. **importFromJson 方法** - JSON 导入
   - 支持数组格式和单个对象格式
   - 自动字段映射

5. **冲突处理逻辑**
   - 根据 ID 检测冲突
   - 支持三种策略处理冲突

6. **AI 分析触发**
   - 可选参数 `triggerAIAnalysis`
   - 导入成功后自动触发 AI 分析任务

**依赖注入**:

```dart
ImportService({
  required IdeaRepository ideaRepository,
  required CategoryRepository categoryRepository,
  required TagRepository tagRepository,
  required AITaskQueue aiTaskQueue,
  required AppLogger logger,
})
```

**ImportIdeasUseCase 导入灵感用例** ✅

| 文件 | 变更类型 | 说明 |
| --- | --- | --- |
| `lib/application/usecases/import_ideas.dart` | 新建 | 导入灵感用例 |

**核心功能**:

1. **ImportFormat 枚举** - 导入格式
   - `excel` - Excel 格式
   - `json` - JSON 格式

2. **execute 方法** - 执行导入
   - 支持选择导入格式（Excel/JSON）
   - 支持冲突策略参数（ConflictStrategy）
   - 支持触发 AI 分析参数
   - Excel 格式支持自定义列名映射
   - 返回 Result<ImportResult>

**依赖注入**:

```dart
ImportIdeasUseCase(ImportService importService, AppLogger logger)
```
