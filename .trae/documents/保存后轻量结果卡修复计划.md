## 一、问题分析

### 1.1 功能描述

"保存后轻量结果卡"是指用户在首页保存灵感后，AI分析完成后在首页显示一个轻量的结果卡片，展示：
- 灵感分类
- 生成的标签
- AI摘要
- "查看详情"按钮

### 1.2 当前实现状态

| 环节 | 状态 | 说明 |
|------|------|------|
| 保存灵感 | ✅ 正常 | 保存到数据库成功 |
| 触发AI分析 | ✅ 正常 | 任务队列正常工作 |
| AI分析执行 | ✅ 正常 | 分析结果正确保存 |
| 结果轮询 | ✅ 正常 | 30秒内获取结果 |
| 状态更新 | ✅ 正常 | lastAnalysis 被正确设置 |
| **显示结果卡** | ❌ **缺失** | 首页未显示 AIResultCard |

### 1.3 核心问题

**首页未连接 AIResultCard 组件**

- `AIResultCard` 组件已完整实现（[ai_result_card.dart](file:///d:/trae/qinglinggan/light_idea/lib/presentation/widgets/common/ai_result_card.dart)）
- `HomeState` 已包含 `lastAnalysis` 字段
- 但 `home_page.dart` 中完全没有使用 `AIResultCard` 组件

#### 步骤3：修改 _buildContentList 方法

**文件**: `lib/presentation/pages/home/home_page.dart`

在灵感列表顶部添加 AIResultCard 显示：

```dart
Widget _buildContentList(bool isDark, HomeState homeState) {
  return ListView.builder(
    padding: const EdgeInsets.only(bottom: 140),
    itemCount: homeState.ideas.length + 
               (homeState.isAnalyzing ? 1 : 0) + 
               (homeState.lastAnalysis != null ? 1 : 0),
    itemBuilder: (context, index) {
      // 优先显示 AI 结果卡
      if (homeState.lastAnalysis != null) {
        if (index == 0) {
          return Padding(
            padding: const EdgeInsets.only(bottom: 12),
            child: AIResultCard(
              categoryName: homeState.lastAnalysis!.category,
              tags: homeState.lastAnalysis!.tags,
              summary: homeState.lastAnalysis!.summary,
              onViewDetail: () {
                // 跳转到灵感详情页
                final ideaId = homeState.lastAnalysis!.ideaId;
                context.pushToIdeaDetail(ideaId.toString());
              },
              onDismiss: () {
                // 关闭结果卡
                ref.read(homeProvider.notifier).clearLastAnalysis();
              },
            ),
          );
        }
        index = index - 1; // 调整索引
      }
      
      // 显示加载占位
      if (index == homeState.ideas.length && homeState.isAnalyzing) {
        return const Padding(
          padding: EdgeInsets.only(bottom: 12),
          child: _InspirationCardShimmer(),
        );
      }
      
      // 显示灵感卡片
      final idea = homeState.ideas[index];
      return _buildIdeaCard(idea, isDark);
    },
  );
}
```
