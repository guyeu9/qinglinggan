# 轻灵感前端界面实施计划

## 项目概述
根据前端原型图和产品设计说明书，实现完整的 Flutter 前端界面。严格按照原型图的视觉设计，实现各界面间的跳转逻辑。

---

## 界面清单与对应关系

| 序号 | 界面名称 | 原型图文件 | 产品设计书章节 | 说明 |
|------|----------|------------|----------------|------|
| 1 | 首页记录页 | `1首页/code.html` | 6.1 首页记录页 | 主界面，时间流列表 |
| 2 | 灵感详情页 | `2灵感详情页/code.html` | 6.6 灵感详情页 | 单条灵感完整展示 |
| 3 | 侧边栏 | `3灵感首页左上角点击三个横展开的侧边栏/code.html` | - | 从首页左侧滑出 |
| 4 | 回收站 | `4回收站/code.html` | 10.1 数据回收站 | 已删除灵感管理 |
| 5 | 数据管理 | `5数据管理/code.html` | 6.9 数据导入/导出 | 导出Excel/数据库 |
| 6 | 设置详情页 | `6设置详情页/code.html` | - | 应用设置 |

```

│  └────┬────┘  └────┬────┘  └─────────┘                         │
│       │            │                                            │
│       ▼            ▼                                            │
│  ┌─────────┐  ┌─────────────┐                                   │
│  │ 侧边栏   │  │ AI灵感中心   │                                   │
```

### 详细跳转规则

| 起点 | 操作 | 目标页面 | 跳转方式 |
|------|------|----------|----------|
| 首页 | 点击左上角菜单图标 | 侧边栏 | 左侧滑出 |
| 首页 | 点击顶部星星图标 | AI灵感中心 | 页面跳转 |
| 首页 | 点击右上角加号 | 新建灵感输入 | 弹窗/展开 |
| 首页 | 点击灵感卡片 | 灵感详情页 | 页面跳转 |
| 侧边栏 | 点击"设置" | 设置详情页 | 页面跳转 |
| 侧边栏 | 点击"AI管理" | AI管理详情页 | 页面跳转 |
| 侧边栏 | 点击"备份导出" | 数据管理页 | 页面跳转 |
| 侧边栏 | 点击"帮助与反馈" | 帮助页 | 页面跳转 |
| 设置页 | 点击"回收站" | 回收站页 | 页面跳转 |
| 所有页面 | 点击返回按钮 | 上一页 | 返回 |
