# 回收站页面实现任务分解

## 任务列表

### 任务 1: 创建页面目录结构
- **描述**: 创建 recycle_bin 页面目录
- **验证方法**: 目录 `lib/presentation/pages/recycle_bin/` 存在

### 任务 2: 实现回收站页面主体
- **描述**: 创建 `recycle_bin_page.dart` 文件，实现完整的回收站页面
- **包含内容**:
  1. 顶部导航栏（返回、标题、清空按钮）
  2. 提示信息栏
  3. 已删除灵感卡片列表
  4. 空状态处理
- **验证方法**: 
  - 页面可以正常编译
  - 所有UI元素按照规范显示
  - 支持暗黑模式

### 任务 3: 实现回收站卡片组件
- **描述**: 在页面内实现 `_RecycleBinCard` 组件
- **包含内容**:
  - 标签和剩余天数显示
  - 内容预览
  - 删除时间
  - 彻底删除和恢复按钮
- **验证方法**:
  - 卡片样式正确
  - 按钮交互正常

### 任务 4: 实现交互逻辑
- **描述**: 实现所有用户交互
- **包含内容**:
  - 返回导航
  - 清空确认对话框
  - 彻底删除确认对话框
  - 恢复操作
- **验证方法**:
  - 所有按钮有响应
  - 对话框正确显示

### 任务 5: 更新记录文档
- **描述**: 在根目录更新 `UPDATE_LOG.md`
- **验证方法**: 文档包含本次变更的详细记录

## 执行顺序
1. 任务 1 -> 任务 2 -> 任务 3 -> 任务 4 -> 任务 5
